# Faker-Trump
 This gem is a port of Hank's Trumpy library that generates fake data.
 
### NOTE

* While Faker generates data at random, returned values are not guaranteed to be unique.

Installing
----------
```bash
gem 'trumpy', '~>2.1'
```
